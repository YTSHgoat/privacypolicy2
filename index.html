Privacy Policy

Effective Date: July 25, 2025

1. Introduction

The Tired Eye Discord Bot ("Bot") is designed for intelligent moderation, automated response, and ambient surveillance within Discord servers. This policy outlines what data is collected, how it is used, and your rights under its domain.

2. What Data We Collect

  • User IDs: Used for enforcement actions like timeouts, bans, and warnings.
  • Message Content: Temporarily scanned for harmful language. Messages are not stored.
  • Reactions: Tracked for role assignment based on predefined server rules.
  • Warnings: Held in memory (RAM only) and purged upon bot restart or shutdown.

3. Data Usage

All data serves one purpose: to protect and moderate. No data is sold, analyzed for marketing, or shared beyond its intended function.

4. Data Storage

The Bot stores no long-term user data. Temporary memory is used only during active bot sessions, and all volatile data is erased upon restart.

5. User Rights

If you feel unjustly targeted or require transparency, contact your server administrator to request information or appeal a bot decision.

6. Contact

For any concerns, summon @cryoytsh via Discord.
