Privacy Policy for Tiredeye Discord Bot

Effective Date: July 25, 2025

1. Introduction

The Tiredeye Discord Bot ("Bot") is developed to help manage and moderate Discord servers with features like message scanning and random timeouts. This policy explains what data we collect and how it is used.

2. What Data We Collect

  • User IDs: Used for moderation actions like timeout, warning, and ban tracking.
  • Temporarily scanned for slurs. Messages are not stored.
  • Reactions: Used for role assignment based on server rules.
  • Warnings: Stored in memory (RAM only) and erased when the bot restarts.

3. Data Usage

Data is only used to moderate and protect the community. No data is sold, shared, or used for analytics.

4. Data Storage

No data is stored long-term. Warnings are only stored temporarily in memory during bot uptime.

5. User Rights

You can contact the server administrator to request information or appeal actions taken by the bot.

6. Contact

For questions, contact @cryoytsh on Discord.
